npm init -> iniciar novo projeto com o Node


## Instalando os pacotes

 -> Instalação do Express, mongoose, ejs e dotenv.

        npm install --save express mongoose ejs dotenv

  -> Instalação do nodemon
        npm install --save-dev nodemon


## Configurando Nodemon
    Inserir a linha no arquivo package.json dentro de scripts
        "start": "nodemon index.js" -> npm start (inicia o nodemon)
